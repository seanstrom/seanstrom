## About Me

* I’m a software engineer focused on crafting interactive systems where design, information, and performance intertwine. Which has led me to explore spatial interfaces and editor-like workflows, built upon enriched data models and expressive interactions.
* I’m particularly interested in tools and systems that help people navigate information, reason about ideas, and work creatively. And I enjoy collaborating with people who care deeply about craft, taste, and thoughtful problem-solving.

## My Setup

Fonts:

- [Iosevka](https://github.com/be5invis/iosevka): My favorite font for editors and the terminal
- [Recursive](https://www.recursive.design): A great variable font for my designs

Design tools:

- [Sketch](https://www.sketch.com): A local-first friendly design tool for my mac
- [Figma](https://www.figma.com/): Tried and true design tool for my designs and handoffs
- [ShapeUp](https://basecamp.com/shapeup): My preferred foundations for a mental framework for shaping ideas into projects and products

Development tools:

- [VSCode](https://github.com/microsoft/vscode): My preferred editor for community extensions and LSP support for new / upcoming programming languages
- [Zed](https://github.com/zed-industries/zed): The editor I reach for when I want speed, taste, and crisp colors.
- [Neovim](https://github.com/neovim/neovim): The light-weight editor of choice when jumping around the files and folders inside the terminal
- [Wezterm](https://github.com/wezterm/wezterm): A rust-based terminal with Lua support that allows for me to use Fennel (lisp) for configuration

Keyboard Layout:

- [Colemak](https://colemak.com): My preferred ergonomic keyboard layout
